# MESAD - Demo Website

Site web demo pour ONG MESAD (Mouvement pour l'Education, la Sante et le Developpement).

## A propos de l'ONG
- Cree le 13 octobre 2001
- ONG ivoirienne reconnue (Recepisse n. 63/MEMAT/DGAT/SDVAC du 21/05/2003)
- Mission: protection de l'enfance et promotion de la jeunesse
- 14 villes couvertes + representation en France depuis 2020
- 500 000+ enfants impactes depuis 2001
- 27 ecoles construites (Projet ECOLE POUR TOUS)

## Technique
- HTML/CSS/JS pur - un seul fichier index.html
- Illustrations SVG vectorielles integrees
- 100% mobile responsive (mobile-first)
- Deploy Netlify automatique

## Note
Site demo a vocation de prospection commerciale.
Les images SVG illustratives doivent etre remplacees par les photos terrain de l'ONG.
